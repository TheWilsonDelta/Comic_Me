How do I create my first storyboard?
  1. Once logged in, navigate to the CREATE page. This will open the Create your own storyboard canvas.



  2. From here, click on the LOAD button to upload an image (from your local computer) to be used as the storyboard scene.



  3. Next, utilize the right-side controls to add text to your storyboard. Once the TEXT control has been selected, you can add text to the scene by simply double-clicking on the area of the storyboard where you would like to add the text. Next, set the font size and font color using the text controls situated on the right-side of the canvas. Look out for even more fun elements to add to your storyboard, for example the ability to add icons and shapes to your scene.
  4. Almost there! Enter your storyboard name (maximum of 40 characters allowed), specify the storyboard target audience by selecting a language and category using the left-side control panel.
  5. Now you would be able to download your created storyboard using the DOWNLOAD button (top right of the screen). Alternatively, click on the SAVE TO SERVER button to save the storyboard to the Novo Nordisk cloud-based library.

How do I access my saved storyboards?
  1. From the Home screen, click on the LIBRARY left-side menu item. This will open the cloud-based storyboard library.



  2. From the library you can download a saved storyboard (click on the related DOWNLOAD link) and/or copy the related storyboard URL (click on the related COPY URL link). This storyboard URL can then be used in your surveys app for creating training quizzes.
